## 프로젝트 소개

<p align="justify">
  야구를 좋아하는 사람들의 커뮤니티 입니다. <br>
  구단별 게시판을 통해 팬들과 소통할 수 있습니다❗ <br>

  현재 세팅된 데이터들은 MSW를 활용한 Mock Data로, 추후 회원 기능과 서버 API 연동 작업을 진행할 계획입니다.
</p>

<br>
